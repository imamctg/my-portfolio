# Md Imam Hossain Portfolio

A personal portfolio website built using **Next.js, React, TypeScript, Redux, and Tailwind CSS**. Showcases projects, skills, and contact functionality for potential clients and employers.

## 🌐 Live Demo
https://my-portfolio-two-ivory.vercel.app/en

## 🛠️ Technology Stack
- **Frontend:** Next.js, React, Redux, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** JWT
- **Deployment:** Vercel (frontend), Render (backend)

## 🔹 Features
- Home, About, Projects, Contact pages
- Responsive UI for mobile, tablet, and desktop
- Project showcase with links to GitHub and live demos
- Contact form integrated for client inquiries
